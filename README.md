# Personal Portfolio Website – README
This project is a responsive personal portfolio website built using HTML, CSS, and Bootstrap 5. It showcases personal information, skills, academic journey, portfolio work, and contact details in a clean and modern UI.
# Features
Responsive Navigation Bar (Sticky Top)
Image Carousel (Hero Section)
About Me Section
Skills Section with Cards & Icons
Academic Timeline Section
Portfolio Gallery with Hover Effects
Contact Section with:
Embedded Google Map
Contact Information
Contact Form
Back-to-Top Button with Smooth Scroll
Fully Responsive Design using Bootstrap
# Technologies Used
HTML5
CSS3
Bootstrap 5
Bootstrap Icons
JavaScript (for interactivity)
